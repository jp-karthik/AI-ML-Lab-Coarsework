#How to execute the program#

------------------------------------------------------------------------------------------------------------------

1) Run the code with input.txt present in the same directory. (where input.txt contains the test case for the program)
   -first line of the input.txt contains 0 or 1 or 2
   -from the second line it contains input grid
2) The output of the program will be printed on the terminal.
   - first line contains number of states explored
   - second line contains length of the path travelled
   - from the third line we can see the path travelled by pac-man in the grid
